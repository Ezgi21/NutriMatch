# 🥗 NutriMatch - Akıllı Sağlık Asistanı

## 🚨 Problem
Dünyada diyabet, çölyak ve hipertansiyon gibi kronik rahatsızlığı olan bireyler, hem market alışverişlerinde hem de dışarıda yemek yerken içerik analizi yapmakta zorlanmaktadır. Restoran menülerindeki gizli içerikleri veya ambalaj üzerindeki karmaşık maddeleri hızlıca değerlendirmek, hata payı yüksek bir süreçtir.

## 💡 Çözüm
NutriMatch, kullanıcıların hem paketli gıda içeriklerini hem de restoran menülerini saniyeler içinde analiz etmesini sağlayan yapay zeka destekli bir beslenme asistanıdır. Llama 3 modeli ile verileri işleyerek "Güvenli", "Dikkatli" veya "Riskli" şeklinde trafik ışığı modeliyle sınıflandırır ve sağlıklı alternatif önerileri sunar.

## 🛠️ Kullanılan Teknolojiler
* **Yapay Zeka:** Llama 3 (Groq Cloud API)
* **Geliştirme:** Antigravity (Vibe Coding & Agent Mode)
* **Altyapı:** Vercel Serverless Functions

## ⚙️ Nasıl Çalıştırılır?
1. Bu repoyu bilgisayarınıza indirin.
2. Vercel üzerinde deployment yaparken `API_KEY` isimli Environment Variable alanına Groq API anahtarınızı ekleyin.
3. Tarayıcınızda uygulamayı açarak analize başlayın.
